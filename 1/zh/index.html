家喵二三事

2022年4月8日 开始销售!

介绍

肩负着某种使命
从天界来到人间的薄荷
在途中却因力竭而倒下

主人公是一位普通的大学生
偶然发现了薄荷并且帮助了她
半强迫地让她待在了家中
虽然有点骄傲自大
但又可爱率真的薄荷十分的惹人喜爱

被一只像小猫一样的
有着猫耳和尾巴的天使叫我“主人”
并且被像宠物一样的薄荷撒娇
感觉生活变得甜蜜而热闹
就这样我们享受着卿卿我我缠缠绵绵的生活

在这样的生活中兽人恶魔『莉莉』
和薄荷的妹妹天使『可可』也乱入了其中
一场欢乐的故事就此展开——!

NAME

薄荷

CAST
飴川紫乃

身为半兽天使的女孩子
长着小猫一样的耳朵和尾巴的天使，拥有天使翅膀却没有天使光环
很介意自己翅膀小的事情，平常都是背着假的玩具翅膀。不擅长使用天使们都会的“魔法”， 还没法做到独当一面。

为了寻找“某个东西”而从天界下凡，苦苦寻而不得，中途倒在了路上被主人公所救起。之后住进了主人公家，把主人公亲切地叫做“主人”，像宠物一般亲近。

性格有点小任性，喜欢恶作刚，为了报答主人公的恩情而努力地想照顺主人公，却总是反过来被照顺。

NAME

莉莉

CAST
立花沙羅

身为半兽恶魔的女孩子
有着动物耳朵和尾巴的恶魔，背上长着对黑色的翅膀。能使用恶魔的能力“消除存在感”(使周围难以察觉到这边)身材很好，经常拿这点来逗薄荷。

怀揣着自己的伙食自己挣的想法，便隐藏起自己的真实身份去打工挣钱。喜欢开心的事情，基本上整天都是乐呵呵的样子，但也会突然展露出认真的模样与人交谈。

NAME

可可

CAST
水野七海

身为半兽天使的女孩子
有着动物耳朵和尾巴的天使，与薄荷不样，还有着“天使之环”。作为天使十分优秀，能自如地使用魔法。但是和薄荷一样 ，仍然还不能独当一面。

姐姐薄荷临行前说去一趟地上，却迟迟没有回来，因为担心所以自己也来到了地上。知道姐姐的目的，正奋力想办法改变姐姐的想法。
